# Stacol / 安光油墨

Company website for **Stacol (安光油墨)** — environmental printing inks.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Default locale is Chinese (`/zh`); English is at `/en`.

## Locales

| Path | Language |
|------|----------|
| `/zh` | 中文 |
| `/en` | English |

Copy lives in `lib/i18n.ts`.
